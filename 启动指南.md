# 工具站启动指南(今晚 1 小时跑通)

## 一、环境准备(只做一次,5 分钟)

### 1. 确认 Node.js 已安装
打开 PowerShell,执行:
```powershell
node --version
```
- 看到类似 `v20.x.x` → 已装,跳过
- 报错"不是命令" → 去 https://nodejs.org/ 下载 LTS 版安装,一路下一步

### 2. 注册智谱 API 拿 Key(免费)
1. 打开 https://open.bigmodel.cn/
2. 手机号注册 + 实名认证
3. 左侧菜单"API Keys" → 创建新 Key
4. 复制 Key(类似 `xxxxxxxx.xxxxxxxx`)

---

## 二、配置项目(5 分钟)

### 1. 进入项目目录
```powershell
cd C:\Users\刘帝伟\Desktop\ai-tools
```

### 2. 创建环境变量文件
```powershell
Copy-Item .env.example .env
notepad .env
```
在打开的记事本里把 `your_key_here` 改成你刚才复制的智谱 Key,保存关闭。

`WECHAT_ID` 等也填一下,没有的暂时留默认就行。

---

## 三、安装依赖(5-10 分钟,第一次较慢)

```powershell
npm install
```

看到一堆 `added xxx packages` 就成功了。
如果报错:
- "npm not found" → Node 没装好,回第一步
- 网络超时 → 用国内镜像:
  ```powershell
  npm config set registry https://registry.npmmirror.com
  npm install
  ```

---

## 四、启动开发服务器(2 分钟)

```powershell
npm run dev
```

看到这样的输出就成功了:
```
   ▲ Next.js 14.2.5
   - Local:        http://localhost:3000
   ✓ Ready in 1200ms
```

**不要关闭这个窗口**,保持运行。

---

## 五、在浏览器访问验证

打开浏览器,访问以下页面逐个验证:

| 页面 | 地址 | 验证点 |
|------|------|--------|
| 首页 | http://localhost:3000 | 看到 3 个工具卡片 + Hero |

**批量文案能生成出文字** = GLM API 调通了 ✅

## 六、把微信二维码换上(5 分钟)

1. 微信"我"→ 点头像 → 二维码 → 保存到桌面
2. 把图片改名为 `wechat-qr.png`
3. 复制到项目里:
   ```powershell

   ```
4. 编辑 `app/contact/page.tsx`,找到占位的 div,替换成:

   (文件顶部加 `import Image from "next/image";`)
5. 浏览器刷新 `/contact` 就能看到你的二维码

## 七、部署到 Vercel(免费,Day 3 做)

### 1. 推到 GitHub
- 在 GitHub 网页创建一个空仓库 `ai-tools`(不要勾 README)
- 回到项目目录:
  ```powershell
  git init
  git add .
  git commit -m "init: ai tools mvp"
  git remote add origin https://github.com/你的用户名/ai-tools.git
  git branch -M main
  git push -u origin main
  ```

### 2. 在 Vercel 部署
1. 打开 https://vercel.com/ → 用 GitHub 登录
2. 点 "Add New" → "Project"
3. 找到 `ai-tools` 仓库 → 点 "Import"
4. 展开 "Environment Variables",加一个:
   - Name: `ZHIPU_API_KEY`
   - Value: 你的智谱 Key
5. 点 "Deploy"
6. 等 1-2 分钟,拿到 `https://ai-tools-xxx.vercel.app`

把这个链接填到:
- 闲鱼店铺简介
- 小红书账号简介
- 朋友圈宣告
- 私信话术里

## 八、常见问题排查

### Q1: `npm run dev` 报 "Cannot find module 'next'"
依赖没装好,重新跑 `npm install`

### Q2: 批量文案报"缺少 ZHIPU_API_KEY"
.env 文件没创建或 Key 没填,回第二步

### Q3: 批量文案报 "401 Unauthorized"
智谱 Key 不对或已过期,回 https://open.bigmodel.cn/ 重新生成

### Q4: 批量文案报 "403" 或额度不足
新用户额度用完了,充值 ¥10 即可继续(够用很久)

### Q5: 网页提取报"抓取失败"
目标网站有反爬或需要登录,换一个普通新闻页面试

### Q6: 端口 3000 被占用
```powershell
npx kill-port 3000
npm run dev
```

### Q7: 部署到 Vercel 后批量文案不工作
99% 是没在 Vercel 后台配置 `ZHIPU_API_KEY` 环境变量,加上重新部署
